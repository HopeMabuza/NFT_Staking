# NFT Staking & Metadata Upgrade

A protocol where users mint a dynamic NFT, stake it to earn ERC20 reward tokens, and spend those tokens to upgrade their NFT's metadata — effectively levelling up their NFT through participation.

Users mint a Galaxy NFT, stake it to accumulate Reward Tokens (RWT) over time, then unstake and burn 100 RWT to permanently upgrade their NFT to new metadata. The longer you stake, the sooner you can upgrade.
